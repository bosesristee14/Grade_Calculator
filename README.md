#  Student Grade Calculator

##  Project Description
This Python program takes a student's name and marks, validates the input, calculates the grade using if-elif-else statements, and displays an encouraging message based on the student's performance.

##  Features
- Accepts student name and marks.
- Validates marks (0–100 only).
- Uses a while loop to handle invalid inputs.
- Calculates grades using if-elif-else.
- Displays encouraging messages for every grade.
- Uses a function for clean and reusable code.

##  Technologies Used
- Python IDLE

##  Grading Criteria

| Marks | Grade |
|--------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| 0-59 | F |
|
